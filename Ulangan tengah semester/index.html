Beranda

Saya baru mulai mendalami dunia pengembangan game dan tertarik untuk mengeksplorasi berbagai aspek di dalamnya. Sebagai pemula, saya masih dalam tahap mempelajari dasar-dasar desain, pemrograman, serta mekanika gameplay. Saat ini, saya berusaha mengasah keterampilan dengan membuat game-game sederhana sebagai langkah awal untuk memahami proses pengembangannya secara lebih mendalam.

Selain berlatih secara mandiri, saya juga mencari kesempatan untuk mengikuti kompetisi berskala kecil guna menguji kemampuan serta mendapatkan pengalaman baru. Melalui ajang tersebut, saya berharap dapat belajar dari tantangan yang ada, memperoleh wawasan dari para developer lain, serta terus berkembang dalam bidang ini. Meskipun perjalanan ini masih panjang, saya berkomitmen untuk terus belajar dan mengembangkan diri agar dapat menjadi seorang developer game yang lebih baik di masa depan.
